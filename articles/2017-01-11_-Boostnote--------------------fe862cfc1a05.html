“Boostnote” 개발자를 위한 데스크탑 노트 앱

Boostnote 는 다양한 언어의 코드조각(snippet)을 작성할 수 있고, 마크다운으로 메모 할 수 있는 심플한 데스크탑 노트 앱 이다.

Boostnote 를 처음 알게 된 계기는 github 에서 돌아다니다가 알게 되었다. 즉, 오픈소스 프로젝트 이다.

일본의 MAISIN&CO. 라는 회사에서 만든 앱으로, 최준영(Rokt33r)님이 가장 많은 커밋을 했다.(거의 다 하신거 같다)

최준영 님의 프로필을 보고 어디서 많이 봤던것 같다는 생각을 했는데 알고보니 얼마전 참석했던 play.node2016 X IBM Developer Connect 행사에서 [최신 트렌드의 Electron과 React 그리고 Webpack으로 만드는 멀티플랫폼 데스크톱 앱!] 이라는 주제로 발표를 하신 분이였다. (발표자료)

Electron에 관해서 내공이 있어 보였는데, Boostnote 앱이 react, electron 기반으로 개발되어 있다. 역시 내공은 그냥 나오는건 아닌것 같다.

Boostnote github 를 보면 프로젝트에 대한 설명과 기술스택, 영감을 받은 서비스 등등이 있고, 그중에 가장 인상깊었던 것은 목표(goal) 이였다.

그냥 글쓰는게 즐거워지셨으면 좋겠어요. 😀
(오픈소스로 남을 것!)

아름답다. 이 글을 포스팅 한 이유 이기도 하다.

정리

  • 코드 및 마크다운 으로 가볍게 메모하기 좋은 데스크탑 앱
  • React, Electron을 학습하면서 많은 참고가 될 수 있다.
  • 오픈소스로 도움이 된다면 기여도 하고 싶다.
